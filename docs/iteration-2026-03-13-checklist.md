# 2026-03-13 迭代验收清单

本文档列出本次迭代已实现的每一项修改，供逐条测试。

## 说明

- 本次改动覆盖：首页项目管理、创意输入、一致性资产、通用分镜列表、广告脚本顺序、底层比例支持。
- 文中“预期结果”按当前实现描述，不包含未开发内容。
- `4:3` 已完整支持到创意输入、广告输入、分镜图生成、提示词约束和本地数据模型。
- 视频供应商请求层当前仍只接受 `16:9 / 9:16`，因此视频生成时如果选择 `4:3`，内部会归一为 `16:9` 发给供应商。这是兼容处理，不是前端遗漏。
